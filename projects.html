Public Repo

I enjoy building in public. It helps me to learn from scratch and also helps other people who might face similar challenges in understanding complex concepts. Here are some projects but sometimes I'm bit lazy to update list, check my github for more alpha :P.

  • alloy-merkle-tree : Merkle Trees implementation in Rust
  • sp1-storage-proof : Storage proof with SP1
  • toji : CLI toolkit for EVM blockchains, retrieving rlp encoded block headers and hashes
  • Rust accumulators : Accumulator library that contains MMR and Merkle Tree implementation
  • L2 indexer : Indexer for Ethereum Layer 2s (OP Stack rollups, Arbitrum, etc...)
  • mini-storage-proofs : Simple full workflow for exploring storage proof concept in evm/starknet/offchain
  • rifle : Starknet Blocktime estimator in Rust
  • chess-dojo : Chess game built on Dojo engine and Cairo language.
